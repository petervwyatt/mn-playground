== Figures

This tests different kinds of figures, esp. SVG since those can cause font related issues. This is some normal text with an inline image image:Letter_T.png[Some alt text for inline] to see how line wrapping goes for sentences that are excessively long and wordy. The next block is for testing PlantUML SVG.

.Testing PlantUML
[plantuml,target="PlantUMLTest",format=svg]
....
@startsalt
skinparam lengthAdjust none
{
  {T!
    ++ <b>H1</b> | .
    +++ <b>Figure</b> <b>ActualText</b>="Cats and Dogs" <b>Placement</b>=<i>Inline</i> | An image representing the words 'Cats and Dogs' 
  }
}
@endsalt
....
